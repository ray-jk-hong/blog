介绍

SystemTap（stap）是一种脚本语言和工具、用于动态检测正在运行的Linux程序甚至是Linux内核。
SystemTap允许用户编写和复用简单的脚本，以深入检查正在运行的 Linux 系统的活动。这些脚本可以快速（且安全地）提取、过滤和汇总数据，从而诊断复杂的性能（甚至功能）问题。
SystemTap脚本的核心思想是命名事件，并赋予其处理程序。当SystemTap运行脚本时，它会监视事件；一旦事件发生，Linux 内核就会将处理程序作为一个快速子程序运行，然后继续执行。
事件有多种类型：进入/退出函数、计时器到期、会话终止等等。处理程序是一系列脚本语言语句，用于指定事件发生时要执行的操作。这些操作通常包括从事件上下文中提取数据，将其存储到内部变量中，并打印结果。

安装

在Debian或者Ubuntu上执行如下命令可以安装：
apt install systemtap linux-headers-$(uname -r)
